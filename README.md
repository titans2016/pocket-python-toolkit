# 🐍 python-utils

A collection of handy Python scripts for everyday tasks — file management, text processing, and more.

## 📦 Scripts

| Script | Description |
|--------|-------------|
| `rename_files.py` | Batch rename files with patterns, numbering, or date prefix |
| `find_duplicates.py` | Find duplicate files in a folder by comparing checksums |
| `text_stats.py` | Count words, lines, chars and top words in a text file |
| `folder_size.py` | Show size of each subfolder sorted by size |
| `json_pretty.py` | Pretty-print and validate any JSON file |

## 🚀 Usage

No dependencies required — pure Python 3.

```bash
python rename_files.py /path/to/folder --prefix "photo_" --numbering
python find_duplicates.py /path/to/folder
python text_stats.py myfile.txt
python folder_size.py /path/to/folder
python json_pretty.py data.json
```

## 🐍 Requirements

- Python 3.7+
